# AI Job Application Assistant

A conversational AI assistant built using Streamlit, LangChain, and Gemini.

## Features

- Resume PDF upload
- Extracts name, email, and skills
- Conversational interaction
- Application summary generation
- Downloadable output

## Tech Stack

- Python
- Streamlit
- LangChain
- Gemini API
- PyMuPDF

## Run locally

```bash
streamlit run app.py
